# Rawrnot - Project Context

## Overview
Rawrnot is a high-fidelity, dual-aesthetic platform designed for the "Primal" and "Obsidian" lifestyles. It integrates high-contrast glassmorphism with textured, skeuomorphic design.

## Technical Stack
- **Frontend:** React (TypeScript) with Tailwind CSS.
- **Backend:** Node.js / Swift (Vapor) on Google Cloud Run.
- **Database:** Cloud SQL (PostgreSQL), Memorystore (Redis).
- **Architecture:** Composable MACH architecture.
- **AI Integration:** Model Context Protocol (MCP) for agentic collaboration.

## Design Principles
- **The Toggle:** Dual CSS variable sets for `obsidian-roar` and `primal-light`.
- **Aesthetics:** "Liquid Glass" vs "Artisanal Pulse".

## Workflow Instructions
- All changes must adhere to the high-fidelity design standards.
- Ensure type safety in all TypeScript components.
- Maintain consistent spacing and typography according to the design system.
